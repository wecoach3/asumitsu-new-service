# asumitsu-new-service

アスミツ新規サービス開発プロジェクト

## 概要

「アスミツ適職診断」を基盤とした新規キャリアサービスの企画・開発リポジトリ。

## 現状の課題

- LINE流入35,349人のうち、マネタイズできているのは2.88%（1,018人）
- 93%のLINEリスト（約33,000人）が未活用
- 収益導線が「面談→エージェント送客」の1本のみ

## 検討中のサービス

1. **履歴書サービス** — yagish型の無料履歴書作成ツール
2. **スカウトサービス** — 年収300万台向けの空白地帯を狙う

## 方針

- 個人課金なし（ユーザーは完全無料）
- 初期マネタイズは送客（エージェント紹介）
- ユーザー数拡大後にB2B課金（企業向けスカウト機能等）
- SNSマーケ（TikTok / Instagram / Threads）で集客
- コンセプト重視（競合の有無より見せ方で勝つ）
