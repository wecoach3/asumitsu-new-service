## 2. C向けサービスの詳細

### 履歴書作成フロー

1. ブラウザまたはアプリでyagishにアクセス
2. テンプレートを選択（用途別・職種別）
3. フォーム入力形式で情報を入力
   - 郵便番号から住所自動入力
   - 生年月日から卒業年月日を自動計算
4. 作成完了後、以下の方法で出力
   - PDF保存
   - コンビニ印刷
   - メール送信
   - ヤギプライムの郵送代行

**最短3分で作成可能**

### 対応書類

- 履歴書: 12種類のテンプレート
- 職務経歴書: 28種類のテンプレート
- 送付状（カバーレター）

### テンプレートの種類

**用途別**:
- 転職用
- 新卒用
- アルバイト用
- パート用
- インターン用

**職種別**:
- 営業
- 企画
- エンジニア
- デザイナー
- 経理
- 看護
- 介護 など

### スマホ対応

- **iOS**: App Storeで「履歴書・職務経歴書 作成 Yagish」として配信
  - 評価: **4.4/5.0**（約3,800件のレビュー）
- **Android**: Google Playで配信
- PC・スマホ・タブレット間でデータをシームレスに同期
- LINEとの連携機能あり

### 無料版と有料版（yagiprime）の機能差

#### 無料版
- 履歴書・職務経歴書の作成
- テンプレート選択
- PDF保存
- 自動バックアップ
- 例文参照
- コンビニ印刷（別途印刷代）

#### yagiprime（月額550円・税込）
- **AI志望動機生成**: 4つの質問に答えるだけで、履歴書情報をAIが解析して志望動機を作成
- **AI面接練習**: 模擬面接を回数無制限で実施
- **AI職歴要約**: 長い職歴を自動要約
- **AI自己分析**（就活応援プラン）: AIとの対話形式で強み・価値観を掘り下げ
- **AI適性検査**（就活応援プラン）: 性格・価値観・ストレス耐性など多角的に分析
- **AI自己PR作成**
- **フォント切り替え**
- **メール便・宅配便サービス**
- **キャリアアドバイザー連携**

**注記**: 2025年11月にヤギプライムの一部サービスが終了しているため、現在の提供内容は変動している可能性あり。

### AI機能の詳細

1. **AI志望動機生成**: 履歴書の内容＋4つの質問回答からAIが最適な志望動機を自動作成
2. **AI面接練習**: 何度でも模擬面接が可能
3. **AI職歴要約**: 冗長な職歴を簡潔にまとめる
4. **AI自己分析**: 対話形式で自分では気づかない強み・価値観を発見
5. **AI適性検査**: 能力・性格・ストレス耐性を多角的分析
6. **AI履歴書読み取り（OCR）**: 2026年2月リリース。既存の紙の履歴書を撮影するだけでデータを自動入力
7. **AIオファー文章自動生成**（企業向け）: ヤギオファーでスカウト文面をAI生成

### コンビニ印刷の仕組み

1. yagishで履歴書を作成
2. eプリントサービスに登録してコンテンツ番号またはQRコードを発行
3. 全国のコンビニ（セブンイレブン、ファミリーマート、ローソン、ミニストップ、デイリーヤマザキ、セイコーマート）の複合機でコンテンツ番号入力・QRスキャン
4. 印刷して料金を支払い

- コンテンツ番号の有効期限: 24時間（前払いの場合は3日間）
- 用紙: L判、2L判（フォト用紙）、A3/A4/B4/B5（普通紙）、A4光沢紙
- カラー印刷: 証明写真付き
- 白黒印刷: 証明写真なし
- 2025年7月からキャッシュレス決済にも対応

### ユーザーの利用シーン（構成比）

※以下は企業向け媒体資料から流通した推定値。yagish公式が一般向けに公開した数字ではない

| セグメント | 割合 |
|------------|------|
| 中途転職 | 約70% |
| アルバイト・パート | 約20% |
| 新卒 | 約10% |

## 3. B2Bサービス

### 3-1. ヤギオファー（ダイレクトリクルーティング）

#### サービス概要
- 2023年9月リリース
- yagishで履歴書を作成した求職者に対して、企業が直接スカウトを送信できる
- 「1万円de採用」がコンセプト
- 毎日約12,000人の求職者がyagishにアクセス

#### 料金体系

| 項目 | 詳細 |
|------|------|
| 基本料金 | 月額10,000円（税別） |
| 初期費用 | なし |
| 成果報酬 | なし |
| 含まれるオファー数 | 月100通 |
| オファー承認者の情報閲覧 | 無制限 |
| チャット機能 | 無制限 |
| 利用期間 | 30日間（自動更新なし） |
| 追加課金 | 10,000円でオファー100通＋30日間延長 |

**雇用区分別に個別課金**: 新卒・中途・アルバイト・複業の各カテゴリーで月額10,000円が必要

**支払い方法**: クレジットカード、請求書、キャンペーンコード、PayPay（導入予定）

#### 導入企業の推移

| 時期 | 企業数 |
|------|--------|
| 2023年11月（リリース2ヶ月） | 500社 |
| 2025年2月 | 2,600社 |
| 2025年6月 | 5,000社超 |
| 2026年（最新） | 6,000社 |

#### 導入企業の業種・事例

| 業種 | 企業例 | 成果 |
|------|--------|------|
| アミューズメント | 一六商事HD | 300通→13承認→1名内定 |
| 介護 | 介護NEXT | 100通→6名反応→1名採用（通常100万→1万円） |
| 医療 | 翔友会 | 看護師・臨床工学技士・医療事務3名採用 |
| 行政書士 | - | 100通→7名レスポンス→4名面接→1名採用 |
| 電気工事 | 東京都内 | 7万円で1名採用 |
| 接客業 | - | 従来40万円/1名→大幅コスト削減 |

#### 企業がユーザーにアプローチする方法
1. キーワード検索で条件に合う候補者を検索
2. 履歴書・職務経歴書の全文を閲覧
3. オファーメッセージを送信（AI自動生成も可能）
4. 求職者が承認すると連絡先が開示
5. チャット機能でやり取り

#### 競合優位性
- **圧倒的低価格**: 月1万円は業界最安水準
- **転職顕在層にリーチ**: 履歴書を「今まさに作っている」人＝転職意欲が最も高い層
- **詳細な職歴情報**: 職務経歴書の全文を見てからスカウトできる
- **全国均等分布**: 自然流入のみで集客しているため、地方にも偏りなくユーザーが存在
- **集客コストゼロ**: 自然流入のみなので低価格が実現可能

## 5. ビジネスモデルの詳細

### 売上規模

- **がっちりマンデー（2025年3月）**: 「5億円ビジネス」として紹介
- 推定年商: 約5億円（2024年度または直近12ヶ月）

### 収益の柱（時系列での進化）

| フェーズ | 時期 | 収益源 |
|----------|------|--------|
| 第1期 | 2018年〜 | コンビニプリント手数料 |
| 第2期 | 2021年〜 | RT送客（転職エージェントへのリード送客） |
| 第3期 | 2023年9月〜 | ヤギオファー（ダイレクトリクルーティング） |
| 第4期 | 2024年7月〜 | yagiprime（個人向けサブスクリプション） |

### 収益構造の推定

#### ヤギオファー（現在の主力と推定）
- 6,000社 x 月額10,000円 = **月間最大6,000万円**
- ただし全社が毎月継続しているわけではないため、実際はスポット利用が多い
- 推定: 常時アクティブ1,500〜2,500社 x 1万円 = 月間1,500〜2,500万円 = **年間1.8〜3億円**

#### RT送客
- 人材紹介会社への送客フィー（1件あたり数千〜数万円）
- 取引先11社以上
- 推定: **年間1〜2億円**

#### yagiprime（個人課金）
- 月額550円
- 登録250万人のうち加入率は低い（推定1〜3%）
- 推定: 2.5〜7.5万人 x 550円 = **月間1,375〜4,125万円**
- ただし2025年11月に一部サービス終了しており、課金ユーザーは減少している可能性

#### コンビニプリント
- 1件あたり数十〜数百円の手数料
- 初期の収益源だが、現在は比重低下

### ユニットエコノミクス（推定）

- **CAC（顧客獲得コスト）**: ほぼゼロ（100%自然流入）
- **LTV**: 
  - 無料ユーザー: 0円（直接的には）→ RT送客や広告で間接収益化
  - yagiprime: 550円 x 平均継続2〜3ヶ月 = 1,100〜1,650円
  - ヤギオファー企業: 10,000円 x 平均3〜6ヶ月 = 30,000〜60,000円
- **CAC≒0のため、LTV/CACは極めて高い**

### yagishが勝っている点

1. **テンプレートの豊富さ**: 40種類で業界最多
2. **AI機能の先進性**: 志望動機生成、面接練習、OCRなど
3. **コンビニ印刷の利便性**: プリンターなしでも即印刷
4. **スマホ完結**: アプリで全工程が可能
5. **スピード**: 最短3分で作成
6. **独立型**: 転職サイトに紐づいていないため、中立的に利用可能
7. **B2Bへの展開**: 履歴書データを活用した独自のリクルーティングサービス

### yagishが負けている点

1. **ブランド力**: doda、リクナビに比べて知名度が劣る
2. **求人紹介機能**: 履歴書作成の先の就職活動は直接カバーしていない
3. **企業との接続**: dodaやリクナビは自社の求人データベースとシームレスに接続
4. **カスタマイズ性**: フォーマットの細かいカスタマイズができない
5. **プライバシー面の不安**: 個人情報に対する懸念の声がある

## 7. ユーザーからの不満点と弱点

### 具体的な不満（口コミ・レビューから）

#### 個人情報関連（最大の不満）
- 「登録後にエージェントからの電話が大量にかかってくる」（App Storeレビュー複数）
- 「個人情報抜かれるんじゃないか不安」（X投稿）
- 開発者回答: 「利用者が同意した場合のみ連絡される」とのことだが、不信感は残る

#### 機能面
- 「フォーマットの種類が限られていて自由度が低い」
- 「細かい修正やカスタマイズがしづらい」
- 「メールで履歴書が送れなかった」
- 「ログイン情報が分からなくなって使えなくなった」
- 「作成した履歴書がシステムトラブルで消失した」

#### コンビニ印刷
- 「コンビニ印刷の料金が高額」という指摘

### サービスとして足りていないもの

1. **求人検索・応募機能**: 履歴書作成の「入口」に留まっており、「その先の就活」は直接カバーしていない
2. **営業組織の不足**: マーケティング会社の性質が強く、豊富なトラフィックやデータベースを有効活用しきれていない（同社自身が課題と認識）
3. **パートナーアライアンスの不足**: RT送客やヤギオファーの販売パートナーの拡大が必要
4. **プライバシー対策の強化**: 個人情報に対するユーザーの不安払拭
5. **データの保全性**: バックアップやデータ消失への対策

### 後発が勝てる可能性のある領域

1. **プライバシーファースト**: 「個人情報を絶対に外部に渡さない」を明確に打ち出す
2. **履歴書作成〜求人応募〜面接のEnd-to-Endカバー**: yagishが弱い「その先」をカバー
3. **カスタマイズ性**: より柔軟なフォーマット編集機能
4. **AI機能の深化**: yagishを超えるAI支援（面接対策、企業別最適化、合格率予測など）
5. **特定セグメント特化**: アルバイト特化、新卒特化、エンジニア特化など
6. **UX/UI**: スマホ体験のさらなる改善
7. **データ活用型の独自ビジネスモデル**: yagishと異なるマネタイズ方法

## 9. 総合分析と示唆

### yagishの強さの本質

- **CAC≒0のビジネスモデル**: 100%自然流入で250万人のユーザーデータベースを構築
- **履歴書＝転職シグナル**: 「今まさに履歴書を作っている人」は最も転職意欲が高い。このインテントデータを企業に売る仕組みが秀逸
- **ネットワーク効果**: ユーザーが増える→ヤギオファーの価値が上がる→企業が増える→収益増→SEO強化→ユーザー増

### 最大の弱点

1. **SEO依存リスク**: Googleアルゴリズム変更で一気に崩壊する可能性
2. **プライバシー問題**: エージェント電話勧誘への不満が蓄積すると信頼毀損
3. **営業力の欠如**: テクノロジー会社であり営業組織が弱い
4. **yagiprime一部終了**: C向け課金モデルが順調でない可能性を示唆
5. **モート（堀）の薄さ**: 技術的な参入障壁は低い。SEOポジション以外の差別化要素が少ない

### 競合サービス参入のチャンス

| 領域 | チャンスの大きさ | 理由 |
|------|-----------------|------|
| プライバシー重視型 | ★★★★★ | yagish最大の弱点を突ける |
| 特定セグメント特化 | ★★★★ | yagishは汎用型。特化型の方がSEOでも勝ちやすい |
| AI深化型 | ★★★★ | 面接対策、企業別最適化など「その先」のAI |
| End-to-End型 | ★★★ | 履歴書作成〜応募〜面接をカバー。ただし大手と競合 |
| B2B低価格型 | ★★ | ヤギオファーの月1万円は既に十分安い |
