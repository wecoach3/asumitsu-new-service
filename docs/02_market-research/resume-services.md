# 履歴書サービス 市場リサーチ

## 日本と海外の履歴書の根本的な違い

### 海外の「Resume」
- フォーマット自由。デザイン・構成・長さ全て自分次第
- 「どう書くか」「どう見せるか」で差がつく
- AI最適化、ATS対策、スコアリングに意味がある
- Resumeの書き方自体がスキル

### 日本の「履歴書」
- JIS規格でフォーマットが決まっている
- 名前、写真、生年月日、学歴、職歴、志望動機、本人希望欄
- 基本的に穴埋め作業
- 差がつくのは「職務経歴書」の方

### 結論
- 海外のResume Builderのモデルをそのまま持ってきても日本では価値が薄い
- ただしビジネスモデルとマーケティング手法は参考になる

## グローバル市場規模

- 履歴書作成ツール市場: 2025年 約88.6億ドル → 2030年 125.5億ドル（CAGR 7.4%）
- AI搭載レジュメビルダー: 2024年 4億ドル → 2032年 18億ドル（CAGR 20%）

## グローバル主要プレイヤー

### Resume.io（4,000万人+）
- **モデル**: フリーミアム（PDF出力で課金）
- **価格**: $2.95トライアル → $29.95/4週
- **UX**: ステップバイステップ穴埋め式、右側にリアルタイムプレビュー
- **AI機能**: 職種別定型文提案、AI要約生成、AIカバーレター
- **マーケ手法**: 恐怖マーケティング（「98%の履歴書は落ちる」）、社会的証明（ライブカウンター）
- **参考になる点**: 恐怖→解決のSNSコンテンツ設計、社会的証明の数字表示

### Zety（SEO最強）
- **モデル**: フリーミアム
- **価格**: $1.95トライアル → $25.95/4週
- **年商**: $1.2M（社員11名）
- **最大の武器**: 職種別SEOコンテンツの大量生産。「[職種] resume example」で検索1位を大量獲得
- **AI基盤**: OpenAI API + 10年分のレジュメデータ
- **参考になる点**: 職種別SEOコンテンツ→ツール誘導のファネル設計。SNSと併用する第2の集客チャネルとして

### Rezi（AI特化・Forbes 2025ベスト・400万人+）
- **モデル**: フリーミアム + 買い切り$149
- **UX**: チャット形式AIで対話しながら履歴書構築
- **AI機能**: 23項目のATSスコアリング（0〜100点）、求人票ペーストで自動最適化
- **参考になる点**: チャット形式（書けない人でも作れる）、スコアリングのゲーミフィケーション

### Teal（求職CRM一体型）
- **モデル**: フリーミアム（無料版でも履歴書作成・DL無制限）
- **価格**: $9/週〜$29/月
- **差別化**: 履歴書ビルダー + 求職管理ツール（Chrome拡張で求人ワンクリック保存、応募ステージ管理）
- **参考になる点**: 無料で本当に使えるものを提供して囲い込む戦略

### Kickresume（800万人）
- **モデル**: フリーミアム
- **価格**: $8〜$24/月
- **AI**: GPT-4.1搭載
- **独自機能**: 履歴書→個人Webサイトにワンクリック変換、1,500件の「実際に受かった人の履歴書」サンプル
- **参考になる点**: 「受かった人のリアルサンプル」コンテンツは日本版で超強力

### Indeed Resume Builder
- **モデル**: 完全無料
- **戦略**: 履歴書ツールは求人プラットフォームへのロックイン（囲い込み）手段
- **参考になる点**: 無料ツール→プラットフォーム囲い込み→企業課金の構造（yagishと同じ）

### Canva Resume Builder
- **モデル**: 完全無料（Canva Pro誘導）
- **問題**: 72%のテンプレートがATS不合格
- **ただし**: 年収300万台の求人を出す中小企業はATS未導入が多く、実は機能する可能性

### LinkedIn Resume Builder
- **モデル**: 基本無料 / Premium $29.99/月（GPT-4.1搭載）
- **特徴**: プロフィール→履歴書ワンクリック変換
- **参考になる点**: 「一度情報入力→何度でも最適化書類が作れる」UX

## 日本の主要プレイヤー

### yagish（累計1,000万人、会員200万人）★最重要参考★
- **モデル**: 基本無料 + yagiprime月額550円 + B2B「ヤギオファー」
- **B2B**: ヤギオファーで7,900社が導入。月額1万円〜
- **構造**: C向け無料で集客 → ユーザーベースの上にB2Bスカウトを載せる
- **集客**: SEO中心
- **学び**: 「無料履歴書ツール→ユーザー集めて→企業にスカウト機能を売る」ハイブリッドモデル

### リクナビNEXT（1,300万会員）
- **モデル**: 完全無料（リクナビNEXT会員登録必要）
- **収益化**: 求人プラットフォームへの送客（B2B求人掲載料）

### doda（834万会員）
- **モデル**: 完全無料（doda会員登録必要）
- **収益化**: エージェント送客 + 求人広告

## ビジネスモデル分類

| タイプ | 代表例 | 日本での適用性 |
|--------|--------|--------------|
| フリーミアム（DL課金） | Resume.io, Zety | △ 個人課金なし方針に合わない |
| 完全無料（送客型） | Indeed, リクナビ, doda | ◎ 御社方針に合致 |
| 無料+B2B二毛作 | yagish | ◎ 最も参考になるモデル |
