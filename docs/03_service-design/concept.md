# アスミツ履歴書 サービスコンセプト

---

## サービス名（仮）

**アスミツ履歴書**

---

## 一言で

「履歴書の書き方がわからない人に、AIが志望動機・自己PRを作ってくれる。無料。」

---

## 1. 誰のためのサービスか

### 対象者

**転職活動中、またはこれから転職しようとしている人。**

具体的にはこういう人：

- 年収300万円台
- 高卒・専門卒
- 20代〜30代前半
- 初めての転職、または転職経験が少ない
- 履歴書の書き方に自信がない

### この人たちはどんな人か

工場勤務、販売職、飲食、介護、事務など、いわゆる「大手の転職サイトがメインで狙わない層」の人たち。大卒でホワイトカラーのキャリアを積んできた人とは、転職活動の事情が全く違う。

**転職活動の特徴：**

| | 大卒ホワイトカラー | この層（高卒/専門卒・年収300万台） |
|--|------------------|-------------------------------|
| 転職期間 | 3〜6ヶ月 | 2週間〜1ヶ月 |
| 選考プロセス | 書類→1次面接→2次面接→最終 | 書類→面接1回→内定 |
| 重視されること | スキル・実績・職歴の一貫性 | 人柄・やる気・すぐ来れるか |
| 履歴書の書き方 | ある程度知っている or エージェントが添削 | **わからない。教えてくれる人もいない** |
| 使うデバイス | PC | **ほぼスマホだけ** |
| 転職の情報源 | 転職サイト・エージェント | **SNS（TikTok/Instagram）** |

求人広告でよく見る「2週間で内定」という訴求が刺さる層でもある。この層の転職はスピードが速い。書類→面接1回→内定が普通で、長々と選考が続くことは少ない。

**「エージェントに相手にされない」感覚がある。** ビズリーチやリクルートダイレクトスカウトは年収600万以上がターゲット。ハイクラス転職のCMを見ても「自分には関係ない」と感じている。実際に、年収300万台向けのスカウトサービスは日本に1つも存在しない（詳細はmarket-research参照）。

**SNSが情報源。** 転職について調べるとき、Google検索より先にTikTokやInstagramで探す人が多い。私たちの適職診断がSNS経由でCPA300円・35,000人を集められているのは、この層の行動パターンにマッチしているから。

### この人たちは今、何に困っているか

履歴書には「志望動機」と「自己PR」という自由記述欄がある。名前や学歴や職歴は欄を埋めるだけだが、**この2つだけは「自分で考えて書く」必要がある。**

高卒・専門卒で転職経験が少ない人にとって、これが最大のハードルになっている。

```
「志望動機って何書けばいいの？」
「自己PRとか言われても思いつかない」
「書き方ググったけど、例文が大卒ハイスペ向けで参考にならない」
「結局わからなくて、転職活動が止まってる」
```

ここで重要なのは、**「めんどくさい」ではなく「そもそもどう書けばいいかわからない」が先にある**ということ。

履歴書を理解している人にとっては「めんどくさい作業」かもしれない。でもこの層にとっては、志望動機も自己PRも**書いた経験がほとんどない**。バイトの面接で履歴書を出したことはあっても、志望動機を真剣に考えて書いた経験はない人が多い。

だから心理の順番は：

```
「書き方がわからない」→「不安」→「自信がない」→「動けない」
```

「めんどくさいから楽にしたい」ではなく、「怖いから助けてほしい」に近い。

### なぜこの人たちをターゲットにするのか

3つの理由がある。

**理由1: 私たちのSNSマーケで最もリーチしやすい層だから。**

アスミツ適職診断のユーザーも年収300万台・高卒/専門卒が多い。TikTok/Instagram/Threadsでこの層にリーチできることは適職診断で証明済み。具体的には、CPA300円で35,000人をLINE登録させた実績がある。

適職診断の競合はリクルート・マイナビ・doda・ミイダスなど大手を含めて多数存在するが、それでもSNSマーケのコンセプトと見せ方で今の数字を出せている。同じ手法を履歴書サービスでも再現できる。

**理由2: この層は転職エージェントに送客しやすいから。**

このサービスに来る人は「転職活動中で書類に苦戦している人」。つまり**まだ転職先が決まっていない**。エージェント紹介を提案したら受け入れやすい。

これは適職診断のユーザー（「なんとなく気になる」だけの人）よりも温度感が高い。適職診断のLINE→面談設定率は6.65%だが、「書類で困っている人」ならこれより高くなるはず。マネタイズに直結するポイント。

**理由3: この層向けのサービスが空白だから。**

履歴書の書き方に困っている高卒/専門卒向けに、AIで志望動機・自己PRを無料で作ってくれる独立したサービスは存在しない。

マイナビやdodaもAI志望動機生成機能を持っているが、全て自社転職サイトの会員限定機能。「マイナビに登録しないと使えない」「dodaに登録しないと使えない」。会員登録なしで誰でも使える独立サービスがない。

yagish（累計1,000万人の履歴書作成サービス）もAI志望動機を提供しているが、月額550円の有料機能。しかもこの有料プランは苦戦して2025年11月に一部終了している。つまり**需要はあるがyagishは課金壁で取りこぼしている**。私たちは最初から無料で提供する。

## 2. 何を提供するか

### サービスの中身

AIが志望動機と自己PRを自動生成してくれる、無料の履歴書作成サービス。

| 機能 | 内容 |
|------|------|
| 履歴書作成 | JIS規格の穴埋め式。基本情報（名前・学歴・職歴）を入力 |
| **AI志望動機生成** | 職種・経歴を入力するだけで、AIが志望動機を3パターン提案。選ぶだけ |
| **AI自己PR生成** | 同上。AIが自己PRを3パターン提案。選ぶだけ |
| PDF保存 | 完成した履歴書をPDFでダウンロード |
| コンビニ印刷 | プリンターがなくてもコンビニで印刷できる |
| 料金 | **全機能完全無料** |

### 既存サービスとの違い

#### vs yagish（累計1,000万人の履歴書作成サービス）

| | yagish | アスミツ履歴書 |
|--|--------|-------------|
| ユーザーの来る理由 | 「履歴書が必要だから」（ツール目的） | 「書き方わからないから助けてほしい」（不安の解決） |
| 訴求 | 「楽に作れる」（利便性） | 「AIが書いてくれる。書類通る」（不安の解消＋成果） |
| 集客チャネル | SEO（100%検索流入。SNSは一切やっていない） | SNS（TikTok/Instagram/Threads） |
| AI志望動機 | 月550円の有料機能（苦戦して2025年11月に一部終了） | **完全無料** |
| プライバシー | 不透明。登録後にエージェントから大量の電話勧誘が来るとの不満が多数 | ユーザーが自分で公開範囲を選べる透明な設計 |

yagishはSEOで「履歴書 作成」と検索する人を集めている。一方、私たちはSNSで集客する。**集客チャネルが被らないので、同じ構造でも共存できる。**

#### vs マイナビ/doda/リクナビ（大手転職サイト）

マイナビ、doda、リクナビも既にAI志望動機生成を無料で提供している。ただし**全て自社転職サイトの会員限定機能**。

| | 大手転職サイト | アスミツ履歴書 |
|--|-------------|-------------|
| 利用条件 | マイナビ/doda等に会員登録が必要 | **会員登録なしでも使える** |
| 位置づけ | 転職サイトの付随機能の1つ | **独立したサービス** |
| 集客 | 自社サイトの会員向け | **SNSで直接バズらせる** |

yagishが成功した構図と同じ。リクナビもdodaも履歴書ツールを持っていたが、yagishは「独立した履歴書サービス」として出して1,000万人集めた。AI志望動機でも同じことが起きる可能性がある。

## 3. どうやって届けるか

### SNSでの訴求

TikTok/Instagram/Threadsで動画コンテンツを配信する。適職診断と同じSNSマーケの手法。

#### ユーザー心理に沿った3段階の訴求

| 段階 | ユーザーの心理 | 訴求の例 |
|------|-------------|---------|
| **入口**（共感） | 書き方わからない、自信ない | 「履歴書の書き方わからない人、AIが全部作ってくれるよ」 |
| **体験**（解決） | 使ってみたらできた | 「自己PRも志望動機もAIが出してくれた。しかも無料」 |
| **成果**（結果） | 書類が通った | 「これで出したら書類通った」「通過率上がった」 |

#### SNSコンテンツの基本ストーリー

1本の動画で入口→体験→成果を伝える：

```
「正直、履歴書の書き方全然わからんかった」
 ↓
「でもこのアプリ使ったらAIが志望動機も自己PRも全部出してくれた」
 ↓
「しかも無料」
 ↓
「これで出したら普通に書類通ったｗ」
```

#### 訴求のバリエーション

**「わからない」を解決する系:**
- 「履歴書の書き方わからんかったけど、AIが全部教えてくれた」
- 「自己PRとか何書けばいいか意味不明だったけど、AIに聞いたら3秒で出てきた」
- 「志望動機の書き方ググるのやめて、AIに作ってもらった方が早かった」

**「自信ない」を解決する系:**
- 「高卒で自己PR何も思いつかなかったけど、AIが出してきた文章が普通にすごかった」
- 「自己PR書く自信なかったけど、AIが自分の経歴からいい感じに作ってくれた」

**成果系:**
- 「これ使ったら書類通過率が上がった。だからみんな使った方がいいよ」
- 「書類で落ちまくってたけどこれ使ってから通るようになった」

**スピード系:**
- 「志望動機も自己PRも30秒で終わった」
- 「もう自分で志望動機書かなくていい時代」

### SNSからサービスへの導線

適職診断と同じ方式。SNSの動画内で**「アスミツ履歴書」でGoogle検索**と訴求し、検索させてサービスに流入させる。

### ユーザー体験フロー（サービスに来てからの流れ）

```
1. 「アスミツ履歴書」でGoogle検索 → サービスにアクセス
2. 基本情報を穴埋め（名前・学歴・職歴）
3. 志望動機 → AIが3パターン提案 → 選ぶだけ
4. 自己PR → AIが3パターン提案 → 選ぶだけ
5. 完成 → PDF保存 / コンビニ印刷
6. ポップアップ:「あなたの経歴から条件が良くなる求人サンプル」
7. 気になる → タップ → エージェント送客
```

### 個人情報の取得は段階的に行う

SNS経由の初見ユーザーにいきなり個人情報を求めると「怪しい」と思われて離脱する。段階的に情報を取得する設計にする。

| ステップ | 何ができるか | 必要な情報 |
|---------|------------|-----------|
| Step 1 | AIの志望動機生成を試す | 個人情報なし（職種・経歴だけ） |
| Step 2 | 履歴書を保存/PDF出力 | 無料アカウント作成（メールアドレス程度） |
| Step 3 | スカウトを受け取る | 経歴を詳しく登録 |

### LINE登録は必須にしない

適職診断は「SNS→LINE登録→結果受け取り」の導線だが、履歴書サービスはツール。ツールを使いたい人に「まずLINE登録して」は邪魔になる。サービス上で直接使える設計にする。yagishもLINE挟んでいない。

## 4. どうやって稼ぐか

### Phase 1（初期）: エージェント送客

履歴書を作った人は転職活動中。書類はできたが、次は「どこに応募するか」。このタイミングでエージェントを紹介する。

**送客の導線：**

```
履歴書完成！
  ↓
ポップアップ:「あなたの経歴から条件が良くなる求人サンプル」
  ↓
求人サンプルを表示
  ↓
「気になる」→ タップ → エージェント送客
```

履歴書が完成した達成感があるタイミングで出すので、押し売り感を抑えられる。

**yagishより送客の転換率が高くなる理由：**

yagishに来る人は「履歴書が必要だから」来ている。応募先は既に決まっていることが多く、スカウトやエージェント紹介は「間に合ってます」になりやすい。

一方、私たちのユーザーは「書類で苦戦していて助けが欲しい人」。まだ転職先が決まっていない。だからエージェント紹介もスカウトも受け入れやすい。

### Phase 2（将来）: B2Bスカウト

ユーザー数が十分に集まったら、企業向けにスカウト機能を開放する。yagishのヤギオファー（月額1万円・6,000社導入）と同じ構造。ただし、Phase 2はユーザーを集めてからの話であり、今の段階では詳細を詰めない。

## 5. 既存事業（適職診断）との関係

アスミツ適職診断と新サービスはカニバらない。**補完関係にある。**

| | 適職診断 | アスミツ履歴書 |
|--|---------|-------------|
| 取る層 | 潜在層（まだ動いてない人） | 顕在層（もう動いてる or 動こうとしている人） |
| 入口の動機 | 「自分に合う仕事って何？」 | 「履歴書の書き方がわからない」 |
| ユーザーの温度感 | 低い（なんとなく気になる） | 高い（転職活動中 or 直前） |
| 送客転換率 | 6.65%（LINE→面談設定） | これより高くなるはず |

**なぜカニバらないか：** 適職診断は「自分を知りたい」人が来る。アスミツ履歴書は「書類を作りたい」人が来る。動機が全く違うので、同じ人が両方に来ても別の目的で使う。

**両方持つメリット：** キャリア領域の「まだ動いていない人」から「もう動いている人」まで全カバーできる。

### 既存LINEリスト（33,000人）の活用

Webサイトが完成してから、既存のLINEリストにテスト配信する。33,000人の中には「適職診断はやったけど転職活動には至っていない」人が大量にいる。この人たちに「履歴書もAIで作れるよ」と案内することで、初速を出せる可能性がある。

## 6. なぜこのサービスをやるのか（背景）

### アスミツ適職診断の課題

アスミツ適職診断はSNSマーケでLINE登録35,349人を獲得している。しかしマネタイズできているのは全体の2.88%（1,018人）のみ。

原因は、収益導線が「面談→エージェント送客」の1本しかなく、「今すぐ転職したい人」しか収益化できていないこと。93%のLINEリスト（約33,000人）が未活用のまま。

### 新サービスで解決すること

適職診断では取れていなかった**「転職活動中の顕在層」**を、別の入口（履歴書）で獲得する。この層は既に転職活動をしているので、エージェント送客の転換率が高い。つまり、新しい収益導線を作る。

## 7. 実現可能性

### AI生成のコスト

| モデル | 志望動機1件あたり | 月10万件生成した場合 |
|--------|----------------|-------------------|
| Gemini 2.0 Flash | 約0.06円 | 約6,000円 |
| GPT-4o-mini | 約0.08円 | 約8,000円 |
| GPT-4o | 約1.4円 | 約14万円 |

**月10万件生成しても6,000〜8,000円。** 完全無料で提供してもコスト的に全く問題ない。

### 技術的な実現性

- LLM APIを呼ぶだけで基本的に動く。開発コストは低い
- 職種・経歴を入力→志望動機を出力するプロンプト設計は既に確立されている
- マイナビ・doda・リクナビ等が既に同様の機能を実現済みで、技術的なリスクはほぼない

### AI品質

- リリース前にテストして品質を確認する
- 「自分で書くよりマシ」レベルで十分。ハイレベルは求めない
- ターゲット層（高卒/専門卒・初めての転職）にとっては、AIが出してくれるだけで大きな価値がある
